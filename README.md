# TextUtils (React)

A React-based text utility application for transforming and analyzing user input.

## Features
- Convert text to uppercase and lowercase
- Capitalize each word
- Remove extra spaces
- Clear text
- Word count and character count

## Tech Stack
- React
- JavaScript (ES6+)
- Bootstrap

## Concepts Covered
- Functional components
- useState hook
- Props and state
- Controlled components
- Event handling
- String manipulation
- Conditional rendering

## Installation & Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Learning Outcome
This project helped reinforce core React concepts and build confidence
in state-driven UI development.
